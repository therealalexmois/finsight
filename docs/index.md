# Документация FinSight

FinSight — REST API (FastAPI) для интеграции с Tinkoff Invest API и предсказания
краткосрочного движения акций. Монорепо из двух сервисов: `finsight_api`
(HTTP-сервис) и `finsight_worker` (Celery-воркер).

## Содержание

- [Архитектура](architecture.md) — фактическая структура кода, слои clean
  architecture, поток управления, DI, интеграция с Tinkoff.
- [RFC: Проектирование системы](rfc/system_design.md) — высокоуровневый замысел
  системы: контекст, проблематика, терминология, проектные решения.
